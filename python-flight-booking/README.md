# Python Flight Booking System - THE BETTER FARE

یہ Python میں بنا ہوا professional flight booking application ہے جو Flask framework استعمال کرتا ہے۔

## Features
- Dynamic flight segments (1-6 flights)
- Multiple passengers (1-6 passengers) 
- Professional PDF e-ticket generation with ReportLab
- THE BETTER FARE branding with logo
- Complete form validation
- Beautiful responsive design

## کیسے چلائیں:

1. Python folder میں جائیں:
   ```bash
   cd python-flight-booking
   ```

2. Application چلائیں:
   ```bash
   python app.py
   ```

3. Browser میں کھولیں: http://localhost:8000

## Files Structure:
- `app.py` - Main Flask application
- `models.py` - Data validation with Pydantic
- `pdf_generator.py` - Professional PDF generation
- `templates/index.html` - Form interface
- `static/style.css` - Beautiful styling
- `static/logo.png` - THE BETTER FARE logo
